### 整体目标
给定地图与市区内的初始点和目的地，确定一条最短的路线，控制汽车的**油门**、**刹车**、**方向盘**，实现自动驾驶功能。

### 任务划分

* 给定百度地图中的数据，利用A\*搜索算法搭建一条最短的路径。
* 横向控制：利用Stanley控制器，对方向盘进行建模，根据路径，利用PD控制模拟汽车的循迹路线
